Как пользоваться git

  1. Добавьте Ваш публичный ключ и создайте роепозиторий на сервере через интерфейс Gitty. Это может занять некоторое время (до минуты после отправки запроса).
  2. Создайте git репозиторий у себя на компютере: 
    mkdir my_repo
cd my_repo
git init
git remote add origin git@имя_сервера:имя_вашего_репозитроия.git
echo -n 'first commit' > README
git add .
git commit -m 'first commit'
git push origin master
  3. Теперь Ваш репозиторий создан и любой пользователь, указанный в поле "пользователи" Вашего репозитория, может получить доступ к данным, отправленным на сервер, используя 
    git clone git@имя_сервера:имя_вашего_репозитроия.git
