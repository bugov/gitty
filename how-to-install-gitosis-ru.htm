Установка Gitosis

  1. Устанавливаем необходимые пакеты: 
    apt-get install python-setuptools git-core openssh-server
  2. Создадим папку, из которой будем устанавливать gitosis. 
    mkdir gitosis-src
cd gitosis-src
  3. Установим gitosis из репозитория разработчиков: 
    git clone git://eagain.net/gitosis.git
cd gitosis
python setup.py install
  4. Создадим пользователя для git 
    adduser --system --shell /bin/sh --gecos 'git version control' --group --disabled-password --home /home/git git
  5. Добавим публичный ключ администратора: 
    sudo -H -u git gitosis-init < путь_к_файлу_id_rsa.pub
  6. Создадим папку для конфигурирования git и клонируем туда настройки по-умолчанию 
    mkdir папка
cd папка
git clone git@localhost:gitosis-admin.git
